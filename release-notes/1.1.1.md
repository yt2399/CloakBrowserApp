# CloakBrowser Desktop 1.1.1

## 更新概览

本版本重新发布 Windows 安装包，重点改进安装流程、发布说明和后续代码签名接入方式。安装包仍由 GitHub Actions 云端构建并发布到 GitHub Release。

## 安装器改进

- Windows 安装包改为交互式安装流程，不再默认一键安装。
- 安装前新增合规与免责协议确认页面，用户必须阅读并接受后才能继续安装。
- 安装过程中允许用户自行选择安装目录。

## 发布流程改进

- Release 发布不再使用 GitHub 自动生成的提交记录。
- 每个版本必须提供 `release-notes/<version>.md` 完整版本说明，缺失时发布流程会失败。
- Release 页面会直接使用对应版本说明文件作为正式更新文档。

## 签名与安全说明

- 保留 GitHub Artifact Attestation，用于记录安装包构建来源。
- 新增 Windows 代码签名证书配置入口，后续配置 `WINDOWS_CSC_LINK` 和 `WINDOWS_CSC_KEY_PASSWORD` 后，云端打包流程可自动签名。
- 当前如果尚未配置商业代码签名证书，浏览器或 Windows 仍可能提示“通常不会下载”或要求用户手动确认。

## 安装说明

- 请从官方 GitHub Release 页面下载 `cloakbrowserdesktop-1.1.1-setup.exe`。
- 如浏览器或系统提示安全确认，请确认来源为官方 Release 页面后再继续。
- 安装时请阅读合规与免责协议，并根据需要选择安装目录。

## 合规提醒

CloakBrowserDesktop 仅可用于合法、合规的学习、研究、测试或经授权场景。不得用于未授权访问、账号滥用、欺诈、攻击、规避平台规则或其他违法违规用途。
