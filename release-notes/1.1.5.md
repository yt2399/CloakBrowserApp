# CloakBrowser Desktop 1.1.5

## 更新概览

本版本修复 macOS 云端打包时误读取 Windows 代码签名变量导致失败的问题。

## macOS 打包修复

- 从 Release job 的全局环境变量中移除 `CSC_LINK`、`CSC_KEY_PASSWORD`、`WIN_CSC_LINK`、`WIN_CSC_KEY_PASSWORD`。
- Windows 代码签名变量现在只会在 Windows 打包命令执行前临时导出。
- macOS 打包配置新增 `identity: null`，明确禁用 macOS 代码签名。
- macOS job 不会再把 Windows 签名证书配置当作 macOS 签名证书解析。

## 保留的发布目标

- Windows x64：NSIS 安装包。
- macOS x64：DMG 与 ZIP 安装包。
- macOS arm64：DMG 与 ZIP 安装包。
- Linux x64：AppImage 安装包。
- Linux arm64：AppImage 安装包。

## 安全说明

当前如果尚未配置对应平台的商业代码签名证书，浏览器、Windows 或 macOS 仍可能提示安全确认。请仅从官方 GitHub Release 页面下载安装包，并根据系统提示自行确认来源。
